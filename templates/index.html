Cara Kerja Sistem LevelFind

Sistem ini menggunakan metode Content-Based Filtering dengan tahapan sebagai berikut:

  1. Text Preprocessing: Membersihkan tag HTML, simbol, dan melakukan normalisasi teks pada judul, genre, serta deskripsi game agar data lebih konsisten.
  2. TF-IDF Vectorization: Mengubah korpus teks menjadi representasi numerik (vektor) berdasarkan bobot kepentingan kata unik dalam dataset.
  3. Cosine Similarity: Menghitung sudut kosinus antara dua vektor untuk menentukan tingkat kemiripan matematis antar judul game.

*Persentase Match mencerminkan seberapa identik elemen cerita, mekanik, dan atmosfer game dibandingkan dengan judul acuan Anda.
